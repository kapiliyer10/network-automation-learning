# Network Automation Learning

Weekly exercises following a structured network automation study plan.

## Topics Covered
- Week 1: Python fundamentals (data structures, file I/O, JSON, YAML, regex)
- Week 2: Netmiko (device connectivity, config push, multi-device polling)

## Setup
1. Clone the repo
2. Copy `.env.example` to `.env` and fill in your credentials
3. Copy `devices.yaml.example` to `devices.yaml` and fill in your device details
4. Install dependencies: `pip install netmiko pyyaml python-dotenv`
